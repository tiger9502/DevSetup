# OpenWRT Setup Instructions for B1300

This document largely follows the AX1800 document. Only differences will be detailed.

## Step 1: Firmware
- The latest B1300 firmware can be found [here](https://openwrt.org/toh/gl.inet/gl-b1300#upgrading_openwrt). This guide assumes version 4.3.25.

## Step 2: Installations and Configurations
- After downloading the same openwrt driver, install this instead:
  ```
  opkg install 19.07.3/ipq40xx/generic/easytether-usb-tiny_0.8.9-5_arm_cortex-a7_neon-vfpv4.ipk
  ```
